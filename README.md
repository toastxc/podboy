<h1 align="center">
  PodboY



</h1>

Podboy (styled podboy or PodboY) is a simple script manager for podman, similar to docker's daemon.


### Testing
```bash
git clone https://github.com/toastxc/podboy.git
cd podboy
cargo b
```

### Installation 
```bash
git clone https://github.com/toastxc/podboy.git
cd podboy
cargo b -r
sudo cp ./target/release/podboy /usr/bin/
```

### Removal
```bash
sudo rm /usr/bin/podboy
```
